# GreenCity UI Automation Tests

Проєкт з автоматизації тестування UI сайту GreenCity.

## Технології
* Python, Selenium WebDriver, Pytest, Allure Report.

## Як запустити
1. Встановити залежності:
   `pip install -r requirements.txt`
2. Запустити тести:
   `pytest --alluredir=allure-results`
3. Сформувати звіт:
   `allure serve allure-results`
